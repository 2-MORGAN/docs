# Se connecter à CNP

CNP propose deux méthodes d'authentification.

## Microsoft Entra ID (recommandé)

Si vous avez un compte `@epita.fr` :

1. Accédez à l'URL CNP fournie par votre encadrant
2. Cliquez sur **Se connecter avec Microsoft**
3. Authentifiez-vous avec votre compte `@epita.fr`
4. Vous êtes automatiquement connecté à CNP

<Note>
  Seuls les comptes du domaine `@epita.fr` sont autorisés. Si votre connexion échoue, vérifiez que vous utilisez le bon compte Microsoft.
</Note>

---

## Connexion admin

Pour les accès administrateurs fournis par votre encadrant :

1. Saisissez votre identifiant admin
2. Saisissez le mot de passe correspondant
3. Cliquez sur **Se connecter**

---

## Durée de session

Votre session dure **24 heures**. Après expiration, vous serez automatiquement redirigé vers la page de connexion.

---

## Problèmes de connexion

| Problème | Solution |
| --- | --- |
| « Domaine non autorisé » | Utilisez votre compte `@epita.fr` |
| Identifiants admin refusés | Contactez votre encadrant |
| Page blanche après connexion | Videz le cache du navigateur et réessayez |
