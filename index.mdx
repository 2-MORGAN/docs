# Bienvenue sur CNP

**CNP** est la plateforme de l'équipe 2Morgan qui vous permet de **déployer automatiquement vos projets GitLab sur Kubernetes**, sans vous soucier de la configuration CI/CD, Docker ou Kubernetes.

En quelques clics, CNP :

- Détecte votre stack technologique (Go, Next.js, NestJS, Spring Boot, Django)
- Configure le pipeline CI/CD GitLab
- Génère le Dockerfile et les fichiers Kubernetes
- Déploie votre application et vous donne une URL d'accès

## Comment ça marche

<Steps>
  <Step title="Vous vous connectez">
    Via votre compte Microsoft EPITA ou avec un accès administrateur.
  </Step>
  <Step title="Vous donnez accès à GitLab">
    En enregistrant un Personal Access Token — CNP l'utilise pour travailler sur vos dépôts.
  </Step>
  <Step title="Vous sélectionnez un dépôt">
    Choisissez le projet GitLab à déployer. CNP détecte automatiquement votre framework.
  </Step>
  <Step title="CNP fait le reste">
    CI/CD configuré, Docker prêt, déploiement sur Kubernetes — vous recevez l'URL de votre service.
  </Step>
</Steps>
